# Sneakers Sales Analysis

## Project Overview
This project analyzes sneaker resale data to understand pricing behavior,
brand-level markups, demand patterns, and factors influencing resale value
in the secondary sneaker market.

## Business Problem
How can historical sneaker resale data be used to support pricing,
inventory, and investment decisions in the sneaker resale market?

## Dataset
- Raw resale transaction data sourced from StockX
- Cleaned and feature-engineered datasets created for analysis and modeling

## Analysis & Methodology
- Data cleaning and feature engineering
- Exploratory data analysis (EDA)
- Comparison of multiple predictive models:
  - Linear Regression (Simple & Full)
  - Log Regression
  - Generalized Linear Model (GLM)
  - Decision Tree
- Model evaluation using R² and RMSE

## Key Insights
- Decision Tree outperformed other models in predicting resale prices
- Significant differences in markup across brands
- Mid-range price sneakers showed the highest ROI
- Brand reputation strongly influences resale value

## Tools Used
- R (data analysis, modeling, visualization)
- Excel (data validation and summaries)

## Repository Structure
- `analysis/` – R scripts for data processing and modeling
- `data/` – Raw and processed datasets
- `visuals/` – Key charts and model performance visuals
- `report/` – Executive report (with appendix) and presentation
